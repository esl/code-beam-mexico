DIVERSIDAD

Orgullosamente Code BEAM Lite Mexico 2023 es incluyente.

Promovemos la diversidad durante el evento y ofrecemos apoyo a todas la comunidades tecnológicas interesadas que podrían tener alguna limitante para participar como por ejemplo grupos indígenas, vulnerables y/o personas con discapacidad.

Ofrecemos lugares gratuitos especiales para grupos vulnerables o no representados en el área de tecnología que tengan alguna limitante económica para participar. Las solicitudes para estos grupos cierran el 3 de febrero del 2023.
